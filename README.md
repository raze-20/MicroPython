# MicroPython Projects

Este repositorio reúne distintos proyectos, pruebas y ejemplos desarrollados con **MicroPython**. La intención es centralizar código, configuraciones y experimentos vinculados al uso de microcontroladores, sensores y automatización ligera.

## Contenido del repositorio

Este repositorio puede incluir, entre otros:

- Scripts y módulos escritos en MicroPython.
- Pruebas de comunicación con sensores y periféricos.
- Ejemplos de entrada y salida digital.
- Código para control de LEDs, botones, pantallas u otros componentes.
- Implementaciones simples de lógica, temporizadores y lectura de datos.
- Archivos relacionados con despliegue en dispositivos como ESP32 o ESP8266.

## Objetivo del repositorio

El propósito es mantener un espacio organizado para:

- Aprender y practicar MicroPython.
- Probar componentes electrónicos y sensores.
- Reutilizar módulos en futuros proyectos.
- Documentar configuraciones y experimentos.

## Requisitos

Dependiendo del proyecto, es posible que necesites:

- **ESP32**, **ESP8266** u otro microcontrolador compatible.
- Entorno para cargar MicroPython (esptool, Thonny, mpremote, etc.).
- Librerías específicas para sensores o periféricos.
